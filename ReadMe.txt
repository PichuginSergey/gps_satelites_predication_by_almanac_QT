Данный репозиторий создан для выполнения практического задания курса "Углубленное программирование на C/C++ " из www.stepik.org.
Данный проект осуществляет расчет видимых спутников GPS, ГЛОНАСС для заданной позиции и времени. 
Расчет координат спутников производится на основе альманахов GPS, ГЛОНАСС.
Задача: Определить количество и номера видимых спутников навигационных спутниковых систем GPS и ГЛОНАСС для заданной точки и заданного момента времени.
Для решения поставленной задачи необходимо наличие альманахов систем GPS и ГЛОНАСС. Данные альманахи содержат в совеем составе параметры, описывающие траектории движения спутников. На их основе можно рассчитать координаты спутников на заданное время. После этого для заданной точки и по рассчитанным координатам спутников производится расчет угла возвышения и азимута данных спутников. Если угол возвышения положительный, то данные спутник виден для заданной точки.
